# Karimoo’s Portfolio

This is my portfolio project built as part of the Coursera course using Microsoft Copilot.

## Features
- **HTML Structure (5 pts):** Semantic sections (`header`, `section`, `footer`) for accessibility and SEO.
- **Accessibility (3 pts):** Alt tags on images, labels tied to inputs, ARIA roles for navigation.
- **CSS Styling (5 pts):** Consistent design across the site with responsive layout.
- **JavaScript Interactivity (5 pts):** Hamburger menu toggle, smooth scrolling, project filter, lightbox, and form validation.
- **Responsive Design (2 pts):** Media queries at 768px, mobile-friendly navigation, flexible project cards.

## Deployment
The site is deployed using **GitHub Pages**:
- Repository created and files uploaded.
- Initial commit message: `Initial commit`.
- GitHub Pages enabled from the repository settings.

`http://username.github.io/repository`
